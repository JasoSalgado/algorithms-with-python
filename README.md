## Black Box List Example

    class Node:
    def __init__(self, value, next=None):
        self.value = value
        self.next = next
    
    L = Node("a", Node("b", Node("c", Node("d"))))
	print(L.value)
    print(L.next.value)
    print(L.next.next.value)
    print(L.next.next.next.value)
